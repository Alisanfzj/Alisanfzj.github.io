ajax怎样上传文件

今天就来聊聊怎样通过ajax上传文件吧。

下面的例子是上传一张图片。

实现该功能需要先引入jQuery插件

html部分代码：
input type=”file” name=”img” id=”pic”
input type=”button” id=”btn” value=”上传”

js部分代码：
$(“#btn”).click(function(){
// 模拟表单对象
var formData = new FormData();
formData.append(“img”,$(“#file”)[0].files[0]);//文件域，上传的是第一张图片
$.ajax({
url:”wenjian.php”,//url地址
type:’post’, //提交方式
data:formData,//提交的数据
processData:false, //jquery是否对你的数据进行预处理
contentType:false,//jquery不要自己修改请求内容类型
dataType:”text”,//数据类型
async:true,//是否异步，true为异步，false为同步
// 成功时的回调函数
success:function(data) {
alert(“上传成功”);
}
});
})

php部分代码：
<?php
if(!empty($_FILES)){
//拷贝文件，tmp_name为临时路径，name为拷贝后的路径
move_uploaded_file($_FILES[‘img’][‘tmp_name’],$_FILES[‘img’][‘name’]);
echo $_FILES[‘img’][‘name’];
}else{
echo “error”;
}
?>

这样就能通过ajax完成一个简单上传文件的方法了。😊😊

float浮动的基本知识

Css的背景设置：

背景简写形式 :官方没有提供固定顺讯

颜色单位：英语单词16进制rgb() hsl() transparent

background-color:red;

backgroundimage:url(“https://photo.tuchong.com/395783/g/20329210.jpg“);

background-repeat:no-repeat;

可以设置像素(支持正负数)，可以设置方向和百分比(left=0%,center=50%,right=100%)

background-position:50%;

多背景,多个属性用逗号隔开

background-color:red;

backgroundimage:url(“https://photo.tuchong.com/395783/g/20329210.jpg"),url("http://tse1.mm.bing.net/th?id=OIP.IbYQdbyj5ZaYJvDLgS4oYAEYEY&w=300&h=300&p=0&pid=1.7"),url(“”);

背景设置：

background-color:transparent;背景颜色为透明

background-image:url()背景图片设置

background-repeat: repeat-x | repeat-y |no-reoeat;背景平铺设置

background-position: x y;背景定位设置

background-size:;背景尺寸设置

float浮动：

float:left|right|none|inhret

1.设置float的元素会脱离文档流，往设置的方向进行浮动，直到遇到父级的边界或者其他的浮动元素就会停止

2.浮动的元素会脱离文档流

3.浮动的元素撑不开父级的高度

清除浮动的方法：

1.给浮动元素的父级设置高度

2.clear:left | right | both

3.伪类清浮动

content:before|after{

  content:””;

 display:block;

clear:both;

}

float属性对块级元素的影响

1.设置浮动之后，块级元素的宽度不再跟随父级宽度，而是内容决定(内容撑开宽度)

2.float元素对行级元素的影响

3.可以设置尺寸以及盒模型
